URL Parameters

Add these parameters to the URL to automatically configure the display:

Basic Parameters
  • ?cycle - Auto-start chart cycling
  • ?fullscreen - Enter fullscreen mode
  • ?cycle&fullscreen - Start cycling in fullscreen
Chart Types
  • ?chart=load - Load, Power & Current
  • ?chart=battery - Battery Level & Runtime
  • ?chart=temperature - Temperature
  • ?chart=voltage - Input/Output Voltage
  • ?chart=cost - Power Cost
  • ?chart=amps - Current Draw
Time Ranges
  • ?hours=1 - Last Hour
  • ?hours=24 - Last 24 Hours
  • ?hours=72 - Last 3 Days
  • ?hours=168 - Last 7 Days
Examples
  • ?chart=amps&cycle - Amps chart with cycling
  • ?chart=load&hours=24&cycle - 24h load data with cycling
  • ?cycle&fullscreen - Fullscreen cycling mode
Tip: You can bookmark these URLs or share them to automatically configure the display exactly how you want it!
